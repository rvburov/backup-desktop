## Установка

### Системные требования
- Python 3.6 или выше
- 100 МБ свободного места
- Разрешения на чтение/запись файлов

### Установка на Windows
```bash
# Скачайте Python с python.org
pip install PyQt5
python backup-app.py
```

### Установка на Linux
```bash
sudo apt update
sudo apt install python3 python3-pip
pip3 install PyQt5
python3 backup-app.py
```

### Установка на macOS
```bash
brew install python3
pip3 install PyQt5
python3 backup_app.py
```

## Интерфейс приложения

### Главное окно
![Главное окно](screenshots/main-window.png)

Приложение состоит из двух основных вкладок:

#### Вкладка "Выбор файлов"
- **Список папок** - управление директориями для копирования
- **Список файлов** - отдельные файлы для резервирования
- **Папка назначения** - выбор места сохранения копий

#### Вкладка "Настройки"
- **Планирование** - настройка расписания копирования
- **Дополнительные настройки** - параметры именования и автозапуска

## Первое использование

### Шаг 1: Добавление источников данных
1. Откройте вкладку "Выбор файлов"
2. Нажмите "Добавить папку" для выбора директорий
3. Используйте "Добавить файлы" для отдельных файлов
4. Убедитесь, что файлы появились в списках

### Шаг 2: Выбор папки назначения
1. Нажмите "Выбрать папку" рядом с полем "Папка сохранения"
2. Выберите или создайте папку для резервных копий
3. Убедитесь, что на диске достаточно места

### Шаг 3: Настройка расписания
1. Перейдите на вкладку "Настройки"
2. Выберите тип периода:
   - **Ежедневно** - копирование каждый день в указанное время
   - **Еженедельно** - копирование в выбранный день недели
   - **Ежемесячно** - копирование в указанный день месяца

### Шаг 4: Запуск копирования
- **Автоматический режим**: Нажмите "Запустить" для работы по расписанию
- **Ручной режим**: Нажмите "Копировать" для немедленного выполнения

## Расширенные настройки

### Настройка автозапуска
Приложение может автоматически запускаться при старте системы:

1. Установите галочку "Запускать приложение при старте системы"
2. Приложение автоматически настроит автозапуск для вашей ОС
3. Для отключения снимите галочку

### Параметры именования файлов
- **Добавить дату к имени** - добавляет временную метку к копируемым файлам
- **Создавать отдельную папку** - создает папку с датой для каждого копирования

### Мониторинг процесса
- **Прогресс-бар** показывает ход копирования больших объемов
- **Лог операций** содержит подробную информацию о всех действиях
- **Статусная строка** отображает следующее запланированное копирование

## Решение проблем

### Общие проблемы

#### Приложение не запускается
**Ошибка**: `ModuleNotFoundError: No module named 'PyQt5'`
**Решение**: Установите PyQt5: `pip install PyQt5`

#### Нет прав доступа
**Ошибка**: `PermissionError: [Errno 13] Permission denied`
**Решение**: Запустите приложение от имени администратора или проверьте права на папки

#### Недостаточно места
**Ошибка**: Сообщение о недостатке свободного места
**Решение**: Освободите место на целевом диске или выберите другой диск

### Проблемы с автозапуском

#### Windows
- Проверьте наличие приложения в списке автозагрузки (Task Manager ? Startup)
- При необходимости настройте вручную через реестр

#### Linux
- Проверьте файл `~/.config/autostart/backupapp.desktop`
- Убедитесь, что файл имеет права на выполнение

#### macOS
- Проверьте настройки в System Preferences ? Users & Groups ? Login Items
- Убедитесь, что приложение добавлено в список

## FAQ

### ❓ Сохраняются ли настройки между запусками?
**Да**, все настройки автоматически сохраняются и восстанавливаются при следующем запуске.

### ❓ Можно ли копировать файлы с сетевых дисков?
**Да**, при условии, что у вас есть соответствующие права доступа.

### ❓ Как остановить автоматическое копирование?
Нажмите кнопку "Остановить" на главной панели управления.

### ❓ Поддерживается ли шифрование данных?
**Нет**, в текущей версии шифрование не реализовано. Для конфиденциальных данных используйте дополнительное шифрование.

### ❓ Как просмотреть историю операций?
Вся история записывается в лог в нижней части главного окна приложения.

### ❓ Что делать если копирование прерывается?
Проверьте лог на наличие ошибок, убедитесь в стабильности подключения к сети (для сетевых путей) и наличии свободного места.

### ❓ Поддерживаются ли симлинки и junction points?
**Да**, приложение корректно обрабатывает символические ссылки и точки соединения.
